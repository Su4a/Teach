ПРОТОКОЛ DNS - Domain Name System
Протокол позволяет IP адреса переводить в доменные адреса
DNS - распределенная база данных где есть ключ и значение
- ключ - доменное имя
- значение - IP адрес

Пример:
5.250.250.242 - ya.ru

Как работает DNS
DNS сервер прописывается в настройках любого хоста (может выдавать оператор, или общедоступные)
Как только мы коннектимся к сайту, это соединение ставится на паузу и хост спрашивает у своего DNS сервера, верни IP для сайта ya.ru. Если такая запись есть, то возвращает IP по имени, если записи нет, то спрашивает у своего корневого DNS сервера (таких серверов всего 12 штук в мире) и они возвращают IP адреса других DNS серверов, ответственных за каждую зону
После этого наш DNS сервер кэширует информацию, чтоб не проделывать весь путь заного
Вовращает IP на наш компьютер и кэширует информацию, чтобы не дергать DNS серверПолучаем соединение с сайтом
