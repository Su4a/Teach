MTU - максимальная единица передачи, максимальный размер пакета, который может быть передан по сети без фрагментации

Для Ethernet это значение составляет 1500 байт

Служебная информация состоит из 4 + 14 байт
С значением MTU в 1500 байт эффективность сети будет 0,98%
Для увеличения эффективности значение MTU можно менять (сетевая карта), делать это лучше по всей сети, чтобы избегать фрагментации
