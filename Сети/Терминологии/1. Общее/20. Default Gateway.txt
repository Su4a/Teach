Настройка, которая нужня для общения между устройствами в разных сетях

Default Gateway - шлюз, должен быть назначен из той же сети что и хост (IP адрес)
Устройству в качестве default gateway должен быть назначен IP роутера, к которому оно подключено
Если между устройствами несколько роутеров, тогда между роутерами необходимо настроить сеть по IP сетей

Когда устрйоство пытается отправить пакет, не входящий в его сеть, он отправляет его на шлюз по умолчанию (Default Gateway)
