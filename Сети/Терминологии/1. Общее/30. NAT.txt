NAT придумали вследствие нехватки IP адресов

-подменяет LAN IP адрес на свой NAT public Source Address в протоколе IPv4

Статичный NAT - вручную вписанный IP public адрес, который будет заменять приватные IP адреса входящие на NAT для отправки в интернет. Также преобразует обратно публичный IP адрес на приватный нужной сети

Порты:
Inside - порт, по которому меняет IP из нашей локальной сети в интернет (Sorce)
Outside - порт, по которому меняет Public IP из интернета на локальный (Distanation)

NAT - обширная технология, которая позволяет гибко настроить соединение сетей и раздачу ИП адресов, также и приватных
