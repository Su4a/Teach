Для экономии IP адресов их разделяют на приватных и публичные

Приватыне диапазоны:
From			To
10.0.0.0		10.255.255.255 - 10.0.0.0/8
172.16.0.0		172.31.255.255 - 182.160.0.0/12
192.168.0.0		192.168.255.255 - 192.168.255.255/16

Адреса, котоыре не входят в приватные и не являются служебными - являются публичными адресами

Публичные IP адреса маршрутизируются в интернете, выдаются организации, централизовано, чтобы не было будлирубщих IP адресов

IANA - организация контролирующая выделение номеров в интернете, также распределяет большие блоки адресов своим дочерним региональным интернет регистраторам

Приватные адреса можно использовать как угодно, ни с кем не согласовывать и тд, но в интернете ходят пакеты только с публичными IP адресами

Для того, чтобы пакет вылетел из LAN в интернет он должен содержать публичный IP адрес в качестве Source и Distanation

NAT ретранслирует LAN адрес в выданный публичный IP адрес
