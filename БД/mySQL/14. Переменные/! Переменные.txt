Переменные - ячейки памяти, которые хранят некие данные

Создание переменной
SET @имя_переменной := значение;

Создание переменной
DECLARE имя_переменной INT DEFAULT 0;

Вывести переменную
SELECT @имя_переменной;

Создание и вывод переменной
SELECT @имя_переменной := значение;

Пример:
SELECT @msrp:=MAX(msrp) FROM products;
