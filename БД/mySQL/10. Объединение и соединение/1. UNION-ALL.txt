UNION [ALL] объединяет две таблицы в одну:
- Каждая инструкция SELECT внутри UNION должна иметь одинаковое количество столбцов
- Столбцы должны иметь схожие тепы данных
- Столбцы в каждой инструкции SELECT должны быть в том же порядке
- ALL - не удаляет повторяющиеся значения

SELECT имя_поля1 FROM имя_таблицы1
UNION [ALL]
SELECT имя_поля1 FROM имя_таблицы2


UNION можно применить в пределах одной таблицы:
SELECT имя_поля1 * 0.1 AS имя_поля2
FROM имя_таблицы WHERE имя_поля2 < 3000
UNION
SELECT имя_поля1 * 0.3 AS имя_поля2
FROM имя_таблицы WHERE имя_поля2 > 3000

Пример:
1,2,3 UNION 3,4,1 = 1,2,3,4
1,2,3 UNION ALL 3,4,1 = 1,2,3,3,4,1
