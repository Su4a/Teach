Сортировка
ASD - по возрастанию
DESC - по убыванию
Если в запросе присутствует группировка, то сортировка идет только после группировки
Ставится в конце запроса

Пример:
SELECT * FROM имя_таблицы
ORDER BY имя_поля [ASC | DESC]

Сложное выражение в качестве сортировки:
SELECT имя_поля1, имя_поля2 FROM имя_таблицы
ORDER BY имя_поля1 * имя_поля2 [ASC | DESC]
