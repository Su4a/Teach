LEAD - Обращается к данным из следующей строки окна (смещение назад)
Функцию можно использовать чтобы сравнивать текущее значение строки со следующим
Имеет 3 параметра:
- Столбец, значение которого необходимо вернуть
- Количество строк для смещения, по умолчанию 1
- Значение, которое необходимо вернуть если после смещения возвращается NULL

Пример:
SELECT Date, Medium, Conversions, 
LEAD(Conversions) OVER(PARTITION BY Date ORDER BY Date) AS 'Lead'
FROM Orders
