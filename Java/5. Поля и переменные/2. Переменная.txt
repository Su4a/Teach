Переменная - переменная метода, имеет область видимости только в данном методе

Переменная хранит в себе записанное значение
Объявлять переменные можно в строку через запятую
Локальная переменная по умолчанию не инициализируется

Создание поля/переменной: 
[тип_данных] [имя переменной] = [начальное значение];
