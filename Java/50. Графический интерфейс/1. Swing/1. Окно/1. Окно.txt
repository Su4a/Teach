Окно - это всегда отдельный поток программы, внутри которого работает бесконечный цикл
В объекте окна существует очередь сообщений, которую цикл опрашивает и выполняет

JFrame - это Swing класс, который отрисовывает окно
Для создания окна нобходимо создать свой класс, унаследовавшись от данного класса
Окно при создании будет наделено какими то свойствами, которые в некоторых случаях можно поменять во время работыприложения
Свойства окна пишутся в конструкторе класса

Дефолтные свойства окна:
- Все окна по умолчанию невидимые 
- Нажатие на крестик по умолчанию делает окно невидимым, а не завершает программу

Свойства окна:
- setDefaultCloseOperation(EXIT_ON_CLOSE) - закрытие приложение при закрытии окна
- setLocation(x, y) - установка локации окна в пикселях
- setSize(x, y) - установка размера окна в пикселях
- setBounds(x, y, width, height) - аналог локации и размера в одном методе
- setTitle("name") - задать имя окну
- setResible(false) - установить запрет на изменение размеров окна
- setVisible(true) - установить видимость окна (пишется всегда последним)










- add(component или panel, компоновщик) - добавить компонент или панель с компонентами в окно
