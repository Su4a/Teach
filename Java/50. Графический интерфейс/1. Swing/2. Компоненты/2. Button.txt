JButton - класс, создания кнопки, наследник класса Component
При создании экземпляра есть возможность сразу в конструкторе задать надпись, которая будет отображаться на кнопке

JButton btnStart = new JButton("имя кнопки")

Если кнопки добавлять без компановщика, они размещаются по центру, занимая все место, перекрывая дргу друга
