При создании компонента у него в новом потоке циклически работает метод paintComponent
Любой компонент умеет перерисовываться
Рисовкой занимается объект Graphics

@Override
protected void paintComponent(Graphics g) {
  super.paintComponent(g);
}


Методы перерисовки компонентов:
paintComponent() - стандартный метод всех компонентов, отрисовывет компонент, работает всегда (поток)
repaint() - делает цикл перерисовки компонента (загружает полностью 1 ядро процессора перерисовкой компонента, лучше слипать поток на 16 мс)
