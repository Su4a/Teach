Модификаторы доступа:
public - область видимости для всех (абсолютный открытый модификатор):

protected - область видимости для пакета и наследников

package - область видимости для всего пакета

private - область видимости для данного объекта, внутри себя (класса) (закрытый модификатор):
Модификатор private определяет доступность только внутри класса, и
предпочтительнее всех
