Исключения - исключительная ситуация, ситуация из ряда выходящая
Исключение - класс
Базовый класс исключения - Throwable
Базовый класс имеет кучу методов для работы с исключениями:
getMessage() - выводит сообщение об ошибке исключения в виде текста
getCause() - причина ошибки исключения

Иерархия исключений:
Throwable делится на 2 вида: Error и Exception
