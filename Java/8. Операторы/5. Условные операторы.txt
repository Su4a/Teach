В условия условных операторов можно добавить логические операторы

1. IF оператор
if (условие) {тело} - выполняет тело если условие верно
if (условие) {тело_1} else {тело_2} - выполняет тело_1 если верно условие, если условие неверно - выполняет тело_2
if (условие_1) {тело_1} else if (условие_2) {тело_2} else {тело_3} - выполняет тело, которое совпало с верным условием, в противном случае выполнит тело_3

2. SWITCH оператор
switch (переменная_1) {case константа_1 : код_1; default : код_2;} - сравнивает переменная_1 с другими значениями(константами) и выполняет тот блок кода, который будет равен первому значению, в противном случае выполнит код_2. Условий сравнивания может быть множество, завершающее условие сработает в том случае, если ни одно условие не верно.
breake - выход из условие (означаем конец)
default - блок кода, который выполняется в любом случае
Оператор работает не со всеми типами данных

3. Тернарный оператор
(условие) ? выражение_истина : выражение_ложь;
