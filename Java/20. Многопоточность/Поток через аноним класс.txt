Запуск потока через анонимный внутренний класс/лямбду:

Использовать анонимные внутренние классы удобно когда вы даете определенную задачу только
одному потоку. В таком случае создавать дополнительные классы в проекте нет никакого смысла.

Этот вариант конечно же полностью аналогичен с реализацией интерфейса Runnable, только выглядит намного короче и не требует создания отдельных классов

public class MainApp {
	public static void main(String[] args) {
		Thread thread = new Thread(new Runnable() {
			@Override
			public void run() {
				for (int i = 0; i < 5; i++) {
					System.out.println(i);
				}
			}
		});
		thread.start();
	}
}
