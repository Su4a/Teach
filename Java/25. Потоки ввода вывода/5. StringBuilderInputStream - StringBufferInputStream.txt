— StringBufferInputStream (deprecated). Иногда бывает удобно работать с текстовой
строкой как с потоком байт. Для этого возможно воспользоваться классом StringBufferInputStream
При создании объекта этого класса необходимо передать конструктору объект String
Символы будут преобразовываться в байты с потерей точности, старший байт отбрасывается
