Самый главный класс - Object, от которого неявно наследуются все другие первоначальные классы (супер-суперкласс по умолчанию)

Класс
Класс - это чертеж, который определяет форму и сущность объекта с полями и методами по которым создаются экземпляры класса (объекты) для их использования
Каждый класс определяет уникальный тип объекта (тип данных) со своими свойствами
Класс определяет новый тип данных
По умолчанию  класс помещается в пустой пакет, если он не указан
Модификатор класса - class
Имя класса пишется с заглавной буквы
Имя public класса должно совпадать с именем файла
Класс может содержать классы, методы и поля
На основании класса создаются экземпляры объектов
Класс может быть абстрактным, что не даст создавать экземпляр этого класса
Класс может быть интерфейсом - interface
Модификаторы доступа классов: public/protected/private/default

Классы могут быть:
Внутренний - класс, объявленный в классе
Вложенный - статический класс, объявленный в классе
Локальные - класс, объявленный в методе
Анонимные - класс, объявленный без имени
