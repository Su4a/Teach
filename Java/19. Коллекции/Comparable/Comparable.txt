Comparable - интерфейс, который служит для описания способа сравнения объектов для их дальнейшего упорядочивания (своя реализация сравнения коллекций по объектам)
В интерфейсе Comparable описан метод compareTo(Object o), который отвечает за сравнение объектов нашего класса

Возвращающие значения метода compareTo(Object o):
Положительное число - текущий объект (this) больше o
Отрицательное число - текущий объект (this) меньше o
Ноль - объекты равны между собой
