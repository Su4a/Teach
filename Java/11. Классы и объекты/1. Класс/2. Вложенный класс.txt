Вложенный класс:
Вложенный класс - внутренний статический класс (статический класс не может ссылаться на нестатические объекты, поля и методы)
Тип1.Тип2(статик) имя1 = new Тип1.Тип2()(статик)

Особенности:
Все свойства класса являются статическими и работать могут только со статическими свойствами внешнего класса
Может существовать в отрыве от внешнего класса
Объект статического вложенного класса не хранит ссылку на конкрентый экземпляр внешнего класса
Объект статического вложенного класса может существовать сам по себе
Можно создать несколько объектов вложенного класса
При создании объекта вложенного класса все также нужно указывать его внешний класс
Статический вложенный класс может обращаться только к статическим полям и методам внешнего класса (при этом неважно какой модификатор доступа имеет статическая переменная во внешнем классе)
