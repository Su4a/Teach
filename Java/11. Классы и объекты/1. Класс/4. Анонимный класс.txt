Анонимный класс:
Анонимный класс - это локальный класс без имени. 
Используется тогда, когда нужно переопределить метод класса или интерфейса.
Класс одновременно объявляется и инициализируется
Такие классы нужны для одноразового использования
Они могут быть объявлены не только в методе, но и внутри аргумента метода
Анонимный класс создается с с помощью интерфейса
Создается безымянный класс, реализующий интерфейс, компилятор увидео такой класс, требует от тебя реализовать все методы интерфейса. И создается объект этого класса

Пример:
Runnable anonRun = new Runnable() {
  @Override
  public void run() {}
};

anonRun.run()
