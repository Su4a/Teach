Объект
Объект - это сущность (предмет) для реализации класса по которому они были созданы, существует только после создания объекта по черчежу класса
Объект имеет состояние(поля) и поведение(открытые методы)
Объект имеет уникальный адрес в памяти в хипе (куче), а ссылка на него хранится в переменной инициализации
При создании объекта создаются и инициализируются поля и методы класса 
Конструктор участвует при создании объекта (дефолтный или свой)

Объявление объекта с инициализацией: - имя_класса имя_объекта = new имя_класса()
Объявление объекта без имени: - new имя_класса()

Правая часть - вызов конструктора класса
Методы и поля объектов используются через их имя_объекта.имя_метода()


Обращения к состоянию класса и их родителей
Операция точка служит для доступа к полям и методам объекта по его имени
Не вызываются напрямую из класса, для реализации вызов происходит из членов класса(методы, блоки кода)
.this - обращение к собственному/текущему объекту полям/методам. Ссылаться на текущий объект (метод ссылается на объект у которого был вызван) 
.super - обращение к объекту ближайшего родителя полям/методам. Ссылаться на родителя вызвавшего его бъект. Обращение к члену суперкласса

Взаимодействие объектов
Взаимодействие объектов между собой происходит за счет передачи объектов в параметры метода в которых производятся манипуляции с переданным объектом и действием метода
Внутри метода, который получил объект на вход, доступен весь функционал этого объекта

Методы объектов
.getClass() - выбрать класс
.getName() - узнать имя класса с пакетом (областью видимости)
.getSimpleName() - узнать имя класса без пакета
