Финализация:
Поля и переменные
Поля и переменные, объявленные модификатором final, превращаются в константу, которая имеет неизменяемое значение
Финализированное поле передается наследникам с теми же неизменяемыми значениями (константой). Финализируется и для всех потомков
Метод
Методы, объявленные модификатором final, не может быть переопределен
Класс
Класс, объявленный модификатором final, не может иметь наследников
