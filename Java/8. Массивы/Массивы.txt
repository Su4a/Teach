Массивы
- совокупность однотипных переменных с общим для обращения к ним именем

Массивы могут работать как с примитивами так и с ссылочными типами данных, тех типов данных, которым он был объявлен
Массив имеет постоянную неизменяемую величину, которой он был инициализирован
Массив является объектом (new - создание объекта)
Индекс элементов ячеек массива начинается с нуля
При создании массива элементы инициализируются по умолчанию 0 и null 
Память под массивы выделяется в куче (хиппе)
.lenght - поле массива, которое хранит его длину (размер)

Объявление и инициализация:
тип[] имя_массива = new тип[размер] или тип имя_массива[] = new тип[размер]

Одномерный массив:
тип[] имя_массива = new тип[размер]
тип[] имя_массива = {...,...,...,}

Двумерный массив: - массив массивов
тип[][] имя_массива = new тип[размер][размер]
тип[][] имя_массива = {...,...,...,} {...,...,...,}

Многомерные массивы:
Нерегулярные массивы - линии массивов разной длины
Многомерные массивы можно объявлять и инициализировать по первому размеру ячейки, оставляя другие пустыми для дальнейших их инициализации
