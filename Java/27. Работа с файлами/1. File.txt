Класс, который позволяет работать с файлами системы и управлять файлами на диске компьютера
Файлом может быть как папка так и сам объект
Для того чтобы управлять содержимым файлов, есть другие классы: FileInputStream, FileOutputStream и другие

File file = new File("file.txt") - создание объекта файла (если задан относительный путь то файл будет создан в директории исполнении программы)
File file = new File(".") - использование файла для директории

Методы:
.listFiles() - возвращает список файлов из указанной папки
.getName() - узнать имя файла с расширением
.isDirectory() - проверить, является ли объект файлом или директорией
.isFile() - узнать размер файла в байтах и его абсолютный путь
.length() - порядковый номер жёсткого диска, на котором расположен файл
.getAbsolutePath() - узнать абсолютный путь файла
.getTotalSpace() - узнать занимаемое мето на диске
.delete() - удалить файл или папку (возвращается тру или фолс)
.exists() - проверить, существует ли такой файл
.getFreeSpace() - узнать свободное место на диске
Фактически у класса File почти все методы дублированы: одна версия
возвращает (и принимает в качестве параметра) String, вторая File

Примеры:
Использования файла для директории:
File folder = new File(".");
for (File file : folder.listFiles())
System.out.println(file.getName());
