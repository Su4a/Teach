URI - унифицированный иденификатор ресурса
URL - унифицированный определитель местонахождения ресурса
URN - унифицированное имя ресурса

Когда программа не пожет найти файл она вбрасывает исключения:
FileNotFoundException -> IOException

При обозначении путей часто используются символы "." (для обозначения
текущей директории) и ".." (для родительской директории).

Работа с файлами открывает потоки/ресурсы, которые необходимо закрывать

Классы работы с файлами:
File - Старая система работы с файлами
Paths, Path, Files, FileSystem - новая система работы с файлами
Вместо единого класса File появились три класса: Paths, Path и Files. Также
появился класс FileSystem, который предоставляет интерфейс к файловой системе
