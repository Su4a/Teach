Функция - блок исполняющего кода вне класса
Метод - блок кода с именем

Метод - глагол с говорящим именем его действия
Метод может быть перегружен - когда имя методов в классе одинаковое, но разная сигнатура (набор параметров)
Имя метода пишется с маленькой буквы в верблюжном регистре
В параметры подаются переменные с указанными типами, которые будут участвовать в теле этого метода: ссылочные по ссылки и примитивные по значению
return - команда, которая выводит из метода с возвращаемым значением (может и без)
Сигнатура метода - имя метода и его параметры (не входит возвращаемое значение)
Нельзя написать два метода с одинаковой сигнатурой
Все аргументы передаются КОПИРОВАНИЕМ

Объявление метода: [модификатор доступа] [тип] [имя] (параметры) {тело}

Модификаторы доступа метода: public/protected/private/default

Осторожно:
Нельзя объявлять метод внутри метода
