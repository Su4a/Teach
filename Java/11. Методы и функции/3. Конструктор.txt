Конструктор - особый метод, имя которого совпадает с именем класса
Конструктор запускается и выполняется сразу, как только создается объект
Т.к. конструктор это метод, он может быть с параметрами(аргументами) и другими функциями предназначенными для инициализации
Конструктор есть всегда по умолчанию до момента создания новых конструкторов, но пустой, в тех случаях, если не написан никакой другой конструктор
Т.к. конструктор метод, он может быть перегружен(множество конструкторов)

<имя_класса>() {}
