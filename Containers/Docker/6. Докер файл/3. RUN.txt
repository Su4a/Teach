Исполнительная команда после запуска docker контейнера внутри контейнера
По сути это обычная консоль команда, которую можно выполнить внутри контейнера

Команда:
RUN <команда> <аргументы>

Пример:
RUN apt-get update - обновить linux
RUN apt-get install nginx -y - установить nginx и согласиться с правилами
