Способ определения исполняемой команды при запуске docker конейнера
Исполнительная команда CMD может быть перезаписана, если при запуске контейнера заданы аргументы командной строки
CMD и ENTRYPOINT могут быть вместе в одном dockerfile

Команда:
CMD <команда в контейнере>
CMD [<команда>, <параметр>] - наиболее правильное напсиание, когда команду с параметром делят

Пример:
CMD echo "Hello my FIRST Docker"
CMD ["echo","Hello my FIRST Docker"]
